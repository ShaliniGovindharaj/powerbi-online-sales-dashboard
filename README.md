# powerbi-online-sales-dashboard
Online Sales Dashboard ‚Äì Power BI Project
** Project Description**

This project is an Online Sales Dashboard created using Power BI.
The main goal is to analyze sales data and show insights in a visual way, so business people can easily understand sales performance.

**Objective**

To analyze online sales data

To identify top-selling categories and states

To understand monthly sales trends

To analyze payment methods and profit

**Dashboard Features**

Category-wise Sales (Electronics, Clothing, Furniture)

Sub-Category Sales (Phones, Chairs, Sarees, etc.)

State-wise Sales Map

Monthly Sales Trend Chart

Payment Mode Analysis (COD, Credit Card, Debit Card, UPI, EMI)

KPIs:

Total Sales

Total Profit

Total Quantity

Profit Margin

Filters: Category, Sub-Category, State, City

** Dataset**

The dataset contains:

Category

Sub-Category

State

City

Sales Amount

Quantity

Profit

Payment Mode

 Dataset file: online_sales_dataset.csv
**
 How to Open the Project****

Download this repository

Open Online_Sales_Dashboard.pbix using Power BI Desktop

Use filters to explore the dashboard

Click charts to see detailed data

 **Key Insights**

Electronics has the highest sales

COD is the most used payment method

Maharashtra and Madhya Pradesh are top sales states

Sales are high in March and November
